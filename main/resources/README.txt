Appointment Scheduler
Serves the purpose of managing and organizing appointments and customers
Author: Cade Fisher
Date: 01/18/2024
Email: cfis208@wgu.edu
Application Version: 1
IDE: IntelliJ IDEA 2023.2.2 (Community Edition)
JDK:Java version 17.0.1
JavaFX: JavaFX-SDK-17.0.1

instructions for application:
    Login using username and password
    Main menu screen will appear, from here you can add, modify, and delete appointments and/or customers.
    The reports page will should you appointments that are able to be filtered by contacts. It will also show you appointments by month and type. Will also display customers and how many appointments they have scheduled.
    You are able to log out from reports and main screen.
 The additional report provided will tell the user how many appointments each customer has scheduled.

MySQL driver version: mysql-connector-java-8.0.25
